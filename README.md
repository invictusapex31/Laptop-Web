## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Usage

1. Visit the homepage and click "Start Building"
2. Login or create an account
3. Use the left sidebar to select components
4. View your laptop in 3D in the center
5. Use the AI assistant to configure via voice or text
6. Check the price summary on the right
7. Add to cart when ready
